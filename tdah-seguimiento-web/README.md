# Seguimiento TDAH (web)

Mini aplicación web (HTML + CSS + JavaScript) para:
- Registrar estudiantes
- Registrar seguimiento diario (atención 1–5, tareas sí/no, interrupciones pocas/muchas)
- Ver gráfica semanal simple (últimos 5 días)
- Exportar reporte semanal en TXT y PDF

## Cómo ejecutar
1. Abre `index.html` con doble clic (en tu navegador).
2. Agrega un estudiante, selecciónalo y guarda registros.

## Datos
Los datos se guardan en tu navegador usando `localStorage`.
